#Usage  
###% Yo.pl [OPTIONS] [USERNAME]  
  
  
--- Input messages example ---  
  % Yo.pl all   		<- yo to all users  
  % Yo.pl count			<- yo count  
  % Yo.pl yo CodeHex		<- yo to single user  
--- Options ---  
    -h          : help  
    -v          : version  
    yo         : yo to single user  
    all         : yo to all users  
    count      : yo count
